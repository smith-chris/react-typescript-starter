# React typescript starter

## How to use

Clone the repo and install local dependencies run:

```sh
yarn
```

--

To develop run:

```sh
yarn start
```

and open [localhost:8080](http://localhost:8080/)

To build production package run:

```sh
yarn run prod
```

and you will find compiled files in `/dist` directory.
